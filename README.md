# Inkwell

Note-taking app with drawing support. Built with Tauri v2.

## Installation

### macOS
```sh
brew install --cask tenorio-labs/tap/inkwell
```
> First time opening: right-click → Open → "Open Anyway"

### Windows
```sh
winget install TenorioLabs.Inkwell
```
> If Windows SmartScreen warns you: click "More info" → "Run anyway"
